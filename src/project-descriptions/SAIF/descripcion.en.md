# SAIF: Administration System and Invetory Management
Cross-platform invoicing, quoting, and inventory tool. It helps small businesses manage invoices that comply with Honduran CAI rules, keep product catalogs in sync, track client credit, and export polished PDFs that can be printed or shared on mobile and web.

## Stack
Expo, React, Supabase, jsPDF, JavaScript, Nodejs, CSS, HTML
