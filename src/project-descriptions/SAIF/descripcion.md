# SAIF: Inventory and Billing Administrative System
Herramienta multiplataforma de facturación, cotización e inventario. Ayuda a las pequeñas empresas a administrar facturas que cumplen con las reglas CAI hondureñas, mantener sincronizados los catálogos de productos, realizar un seguimiento del crédito de los clientes y exportar archivos PDF pulidos que se pueden imprimir o compartir en dispositivos móviles y web.

## Stack
Expo, React, Supabase, jsPDF, JavaScript, Nodejs, CSS, HTML
